What should the menu bar item show?

This is the always-visible element in the macOS menu bar. Pick the style that best fits how you want to monitor your deployments at a glance.

⬡ my-site → building…

A — Scrolling ticker

Animates through active deployments. Shows site name + current status. When idle, collapses to just an icon.

⬡ 3 sites 1 building

B — Count + status badge

Shows total sites monitored and a badge for any active or failed deploys. Compact and always readable.

⬡

C — Icon only

Just a Netlify-style icon that changes color: green (all good), yellow (building), red (failed). Minimal clutter.

You can click a card to select it, then let me know in the terminal.
